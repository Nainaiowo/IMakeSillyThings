# IMakeSillyThings

Custom Dalamud plugin repository for Nainaiowo plugins.

Add this custom plugin repository URL in Dalamud:

```text
https://raw.githubusercontent.com/Nainaiowo/IMakeSillyThings/refs/heads/main/repo.json
```

## Projects

- [Chibi Chaos](https://github.com/Nainaiowo/chibi-chaos) - Scale down Chaos.
- [Better Deaths](https://github.com/Nainaiowo/better-deaths) - A clearer party death review.
- [Nai Debug Console](https://github.com/Nainaiowo/nai-debug-console) - Local debug tools for troubleshooting.
- [Honorific Cycle](https://github.com/Nainaiowo/honorific-cycle) - Cycle custom Honorific titles.
